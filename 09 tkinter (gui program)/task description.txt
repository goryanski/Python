1) Должен быть файл с перечнем пользователей + их логины и пароль.
2) Возможность регистрации нового пользователя (добавлятся запись пользователя в файл)
3) Возможность логина пользователя (проверка существования пользователя из файла)
4) Если логин не удачный --> Окно с oшибкой
5) Если логин не удачный 3 раза --> Окно с ошибкой + закрыть программу
6) Если удачный логин --> Окно с большим вводом текста + кнопка сохранить + кнопка открыть или меню с теми же функциями
7) Кнопка сохранить сохраняет весь текст в файл а открыть вставляет текст из файла в поле ввода
8) В меню функции закрыть приложение + разлогиниться
9) При выходе из пользователя и закрытии программы сделать так что бы созадвалась резервная копия созданных файлов в виде архива.
